# Key Capabilities of NestJS

## Application Types
- REST APIs
- GraphQL APIs
- WebSockets gateways
- Event-driven and microservice transports

## Core Runtime Features
- Modules (`@Module`)
- Controllers (`@Controller`)
- Providers/Services (`@Injectable`)
- Pipes (validation/transformation)
- Guards (authorization/authentication flow)
- Interceptors (cross-cutting concerns)
- Exception filters (error shaping)

## Platform Flexibility
- Express adapter (default)
- Fastify adapter (high-performance alternative)
