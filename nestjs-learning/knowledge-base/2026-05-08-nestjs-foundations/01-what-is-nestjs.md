# What Is NestJS

NestJS is a progressive Node.js framework for building server-side applications.

It is TypeScript-first (with JavaScript support) and provides a structured architecture out of the box using modules, dependency injection, and decorators.

At runtime, Nest can use Express (default) or Fastify as the HTTP platform.

## One-line Definition
NestJS is an architecture-first backend framework for Node.js focused on maintainability, scalability, and testability.
